# 安装

## 前置要求

- [Node.js](https://nodejs.org/zh-cn) 18 及以上版本。
- [Git](https://git-scm.com/)。
- 支持 [Markdown](https://markdown.p2hp.com/) 语法的编辑器。
    - 推荐使用 [VSCode](https://code.visualstudio.com/) 。


## 安装 Fispo
所有必备的应用程序安装完成后，即可使用 npm 安装 Fispo 的初始化工具包
``` sh
npm install -g create-fispo
```

安装 Fispo 完成后，请执行下列命令，Fispo 将会在指定文件夹中新建所需要的文件。

``` sh
create-fispo <folder>
```


## 文件结构
初始化后，您的项目文件夹将如下所示：

``` md
.
└── docs                     # 存放博客相关内容
    ├── post                 # 文章目录（Markdown/MDX）
    └── public               # 存放公共资源文件的目录
├── .gitignore
├── .npmrc
├── fispo.config.ts          # 配置文件
├── package.json
├── README.md
└── package.json
```

## 配置文件

配置文件 `.fispo.config.ts` 让你能够自定义 Fispo 站点的各个方面，最基本的选项是站点的标题和描述：
``` ts
import { defineConfig } from "fispo-core";

export default defineConfig({
  title: "XXX17的个人博客",
  author: "XXX17",
  description: "学无止境",
  // 主题配置
  themeConfig: {
    navMenus: [
      {
        title: "首页",
        path: "/",
        icon: "home",
      },
      {
        title: "标签",
        path: "/tag",
        icon: "tag",
      },
      {
        title: "分类",
        path: "/category",
        icon: "folder-open",
      },
      {
        title: "关于",
        path: "/about",
        icon: "heart",
      },
    ],
  },
});
```
还可以通过 `themeConfig` 选项配置主题的行为。有关所有配置选项的完整详细信息，请参见[配置参考]()。

## 启动项目

该工具还应该将以下 `npm` 脚本注入到 `package.json` 中：
``` json
{
  ...
  "scripts": {
    "dev": "fispo dev",
    "build": "fispo build",
    "preview": "fispo preview",
    "deploy": "fispo deploy"
  },
  ...
}
```

`dev` 脚本将启动具有即时热更新的本地开发服务器。使用以下命令运行它：
``` sh
npm run dev
```
除了 `npm` 脚本，还可以直接调用 `Fispo`:
``` sh
npx fispo dev
```
开发服务器默认会运行在 `http://localhost:5173` 上。在浏览器中访问 `URL` 以查看新站点的运行情况吧！
