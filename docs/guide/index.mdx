# Fispo - 现代化博客构建工具

Fispo（全称 Fish in the pool）是一款基于 **React + Vite + TypeScript** 的静态站点生成器，专为开发者和内容创作者打造。通过提供**主题系统**、**插件生态**和**Markdown扩展**等核心能力，帮助用户快速构建高性能、可扩展的现代化博客系统。

## ✨ 核心特性

### ⚡ 极速开发体验
- 基于Vite 4.x引擎，实现**按需编译**和**HMR热更新**
- 首屏加载速度提升40%（对比传统SSG方案）
- 支持现代浏览器原生ES模块导入

### 🎨 主题系统
- 完整的React组件化主题架构
- 支持动态主题切换
- 内置主题市场支持一键安装

### 🔌 插件生态
- 内置插件市场管理系统
- 支持自定义插件开发（TypeScript接口）

### 📝 增强型Markdown
- 完全支持MDX语法（嵌入React组件）
- 自定义代码块样式

### 🛠 工程化能力
- 完整的TypeScript类型定义
